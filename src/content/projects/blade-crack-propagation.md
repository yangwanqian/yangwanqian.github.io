This project develops an OpenFAST-Abaqus coupled simulation workflow for blade crack propagation prediction in large wind turbines.

The work aims to construct a simulation-informed crack database, quantify stiffness degradation under different crack states, and build damage-index regression models for monitoring and prediction.
